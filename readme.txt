Add Custom Stock Statuses for WooCommerce's Inventory Manager.

== Description ==

Custom Stock Status for WooCommerce is asmall, rock solid, simple and **free solution** for a specific task only:
Create as many **Custom Stock Statuses** for ** WooCommerce** Inventory Manager.

= Features =
* Create unlimited number of stock statuses.
* Free plugin without any kind of limitation.
* Work with variable products too.
* Fully **Translateable**
* No CSS, no JS to **not causes any conflict** with the theme.

== Installation ==
= Install from WordPress Admin =
1.  Navigate into the WordPress Admin Plugins menu, than click the Add New button.
2.  Search "Weart Category Posts Widget".
3.  Click Install, than activate.
4.  Go to WooCommerce / Stock Status.
5.  Now you can setup your custom statuses in WooCommerce

= Install with the Manual way =
1.  Upload the plugin into the the WordPress Plugins.
2.  Activate the plugin.
3.  Go to WooCommerce / Stock Status.
4.  Now you can setup your custom statuses in WooCommerce
